# OpenCV Project
A C++ application made to demonstrate computer vision techniques using **OpenCV (v. 4.10)** library. Contains both static images and videos.

## Prerequisites
This project is made for **Windows** environment.

* **Visual Studio (2019/2022)**
* **OpenCV library (v. 4.10)** setup in C:\opencv (or change preferences in project settings)

## Build and run
* Ensure the library is properly linked in project settings.
* Put **dll** files into root and Debug folder
* Build using **F5** in Visual Studio
